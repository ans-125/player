# تعليمات التشغيل السريع

## خطوات التشغيل

1. **تثبيت المتطلبات**:
   ```bash
   pip install -r requirements.txt
   ```

2. **تطبيق هجرات قاعدة البيانات**:
   ```bash
   python manage.py migrate
   ```

3. **تشغيل الخادم**:
   ```bash
   python manage.py runserver 0.0.0.0:8000
   ```

4. **فتح التطبيق**:
   افتح المتصفح وانتقل إلى: http://localhost:8000
